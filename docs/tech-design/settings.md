## Settings

- **Theme:** Dark mode / Light mode toggle
- **Font settings:**
  - Font family selection (monospace fonts for code)
  - Font size adjustment (small, medium, large, custom)
- **Color scheme:** Customizable colors for error/success highlighting

#### Typing Behavior

## Typing Display States

#### Normal State (Default)
- Type complete text from start to finish
- Real-time statistics display (WPM, CPM, accuracy, timer)
- Virtual keyboard with finger mapping and highlighting
- Immediate error highlighting (red background until correct key pressed)
- Smooth highlighting for correctly typed characters (green fade)
- Session summary at completion

#### Zen Mode
- Distraction-free typing environment
- Keyboard and statistics hidden (toggle via settings or shortcut)
- Pure focus on typing without visual clutter
- Error highlighting still works (red/green feedback)
- Statistics tracked in background (viewable after session)

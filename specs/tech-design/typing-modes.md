## Typing Modes

#### Mode 1: Normal Mode (Default)
- Type complete text from start to finish
- Real-time statistics display (WPM, CPM, accuracy, timer)
- Immediate error highlighting (red background until correct key pressed)
- Smooth highlighting for correctly typed characters (green fade)
- Session summary at completion

#### Mode 2: Zen Mode
- Distraction-free typing environment
- No timer, no statistics display
- No pressure, pure focus on typing
- Optional: minimal visual feedback only
- Statistics still tracked in background (viewable after session)
